# Dialog

A **modal dialog** built on the native `<dialog>` element.

## Renders in the top layer

Calling `showModal()` promotes the dialog to the browser **top layer** and gives you,
for free: a **focus trap**, **Esc-to-close**, an inert (non-interactive) background,
and a `::backdrop` pseudo-element for the page dimming. `Dialog` adds **focus return** —
on close, focus goes back to the element that was focused before it opened (§7.6).

No floating-ui here: a modal is centered by the platform, so there is nothing to
anchor or position.

## Fallback

When the native modal API is unavailable, the content still renders but without the
native focus trap; the component degrades rather than throwing.
