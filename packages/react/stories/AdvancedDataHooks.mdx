# Data Hooks

Two hooks expose the raw view model data as render-ready arrays so you can loop over
them in **fully custom headless views**. The built-in view components call these
internally; they are exported so you can use them when building your own layout from
scratch with `useMonthView` or `useAgendaView`.

Both hooks must be called inside a component that lives inside a `CalendarProvider`.
Both return `null` when a different view is currently active, so your component can
return `null` early and bow out cleanly.

---

## `useMonthWeeks`

Returns the month grid as a typed, render-ready structure. Use this when you want to
build a custom month layout — for example, rendering the grid as a `<table>` or
applying a completely different visual arrangement — without reimplementing the
layout algorithm.

```tsx
import { useMonthWeeks, EventButton } from '@big-calendar/react'

function MyMonthGrid() {
  const grid = useMonthWeeks()
  if (grid === null) return null  // not the active view

  return (
    <div>
      <div className="weekday-row">
        {grid.weekdays.map((wd) => (
          <span key={wd.day}>{wd.short}</span>
        ))}
      </div>
      {grid.weeks.map((week) => (
        <div key={week.key} className="week-row">
          {week.days.map((day) => (
            <div key={day.day} data-today={day.isToday}>
              {day.label}
              {day.extra && <button>+{day.extra.count} more</button>}
            </div>
          ))}
          {week.segments.map((seg) => (
            <EventButton key={seg.key} event={seg.event} title={seg.title} resizeEdges={[]}>
              {seg.title}
            </EventButton>
          ))}
        </div>
      ))}
    </div>
  )
}
```

## `useAgendaRows`

Returns the agenda view model as a flat list of day groups, each containing its
resolved events. Use this when you want to build a fully custom agenda layout.

```tsx
import { useAgendaRows, AgendaEventButton } from '@big-calendar/react'

function MyAgenda() {
  const rows = useAgendaRows()
  if (rows === null) return null  // not the active view

  return (
    <div>
      {rows.map((row) => (
        <section key={row.day}>
          <h2>{row.label}</h2>
          {row.events.map((item) => (
            <div key={item.key}>
              <span>{item.allDay ? 'All day' : item.time}</span>
              <AgendaEventButton event={item.event} title={item.title} />
            </div>
          ))}
        </section>
      ))}
    </div>
  )
}
```
