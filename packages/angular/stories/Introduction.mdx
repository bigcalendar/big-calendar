# Big Calendar — Angular

Big Calendar is a framework-agnostic calendar engine with a thin Angular adapter. This Storybook documents the `@big-calendar/angular` package.

## Architecture

`@big-calendar/angular` is a thin Angular 21+ layer over the shared `@big-calendar/core` engine. It mirrors the React and Vue adapters' public API — views, inject composables, ng-template slots, and DnD — adapted to Angular idioms:

- **`inject()` + `InjectionToken`** instead of React context or Vue provide/inject
- **`injectMonthView()` / `injectTimeGridView()` / `injectAgendaView()`** instead of React hooks or Vue composables
- **`ng-template` inputs** (`[bcMonthEvent]`, `[bcTimeDayHeading]`, etc.) instead of React render props or Vue slots
- **`CalendarDndDirective`** applied as `calendarDnd` attribute instead of a composable
- **Angular `Signal`** for reactive state bridging from preact signals

## Quick start

```bash
npm install @big-calendar/angular @big-calendar/localizer-temporal @big-calendar/styles
```

```ts
// app.component.ts
import { Component } from '@angular/core'
import { createTemporalLocalizer } from '@big-calendar/localizer-temporal'
import { CalendarProviderComponent } from '@big-calendar/angular'
import { CalendarComponent } from '@big-calendar/angular'
import '@big-calendar/styles/index.css'

@Component({
  standalone: true,
  imports: [CalendarProviderComponent, CalendarComponent],
  template: `
    <bc-calendar-provider [localizer]="localizer" [events]="events" defaultView="week">
      <bc-calendar />
    </bc-calendar-provider>
  `,
})
export class AppComponent {
  readonly localizer = await createTemporalLocalizer({ locale: 'en-US', timeZone: 'UTC' })
  readonly events = []
}
```
