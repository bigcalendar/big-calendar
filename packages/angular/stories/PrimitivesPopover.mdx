# Popover

`BcPopoverComponent` (`<bc-popover>`) is a floating panel primitive used internally by Big Calendar's show-more overflow buttons. You can also use it directly in custom slot templates to build your own event detail popovers, date pickers, or any anchored overlay.

## Architecture

`BcPopoverComponent` uses Angular CDK Overlay together with `@floating-ui/dom` for positioning. The component manages open/close state through the browser's native Popover API when available, with a fallback for older browsers.

## Basic usage

The popover is split into two parts:

1. A trigger element decorated with `[bcPopoverTrigger]` — the button that opens the popover.
2. A `<bc-popover>` element that holds the floating panel content.

```html
<button type="button" [bcPopoverTrigger]="myPopover">Open popover</button>

<bc-popover #myPopover placement="bottom-start">
  <p>Popover content goes here.</p>
</bc-popover>
```

## Using with a custom show-more slot

The internal show-more overlay in the calendar uses `<bc-popover>` internally. When you replace `[bcMonthShowMore]`, you can use `<bc-popover>` to keep the same anchored-overlay behaviour:

```html
<ng-template #customShowMore let-ctx>
  <button type="button" #trigger [bcPopoverTrigger]="pop">
    {{ ctx.count }} more
  </button>
  <bc-popover #pop placement="bottom-start" [sameWidth]="true" class="bc-popover bc-show-more-popover">
    @for (item of ctx.events; track item.key) {
      <div class="bc-segment">{{ item.title }}</div>
    }
  </bc-popover>
</ng-template>
```
