# Tooltip

`BcTooltipElement` (`<bc-tooltip>`) is the tooltip primitive used internally by Big Calendar to show event titles when they overflow their container. You can use it directly in custom elements to show any content on hover or focus.

## Architecture

`BcTooltipElement` uses `@floating-ui/dom` for positioning. It opens on `mouseenter` / `focusin` and closes on `mouseleave` / `focusout`.

## Basic usage

Wrap the element you want to trigger the tooltip, and add the content inside `<bc-tooltip>`:

```ts
html`
  <bc-tooltip>
    <button type="button" slot="trigger">Hover me</button>
    <span slot="content">This is a tooltip</span>
  </bc-tooltip>
`
```

## Accessibility

`BcTooltipElement` automatically:

- Assigns a unique `id` to the tooltip panel.
- Sets `aria-describedby` on the trigger element pointing to that `id`.
- Hides the tooltip panel from the accessibility tree when closed (`aria-hidden="true"`).

This means screen readers announce the tooltip content when the trigger receives focus, without requiring any extra markup from you.

## Using in a custom element

```ts
@customElement('my-truncated-event')
class MyTruncatedEvent extends LitElement {
  override createRenderRoot() { return this }

  @property() title = ''

  override render() {
    return html`
      <bc-tooltip .content=${this.title}>
        <span
          slot="trigger"
          style="overflow:hidden;text-overflow:ellipsis;white-space:nowrap;display:block"
        >
          ${this.title}
        </span>
      </bc-tooltip>
    `
  }
}
```
