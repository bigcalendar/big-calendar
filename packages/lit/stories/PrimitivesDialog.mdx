# Dialog

`BcDialogElement` (`<bc-dialog>`) wraps the native `<dialog>` HTML element. It uses `showModal()` to display the dialog as a modal overlay with a backdrop, and `close()` to dismiss it. All browser-native focus trapping and `Escape` key handling are included automatically.

## Basic usage

```ts
html`
  <bc-dialog
    .open=${isOpen}
    .onBcClose=${() => { isOpen = false; requestUpdate() }}
    aria-label="Event details"
  >
    <h2>Meeting with Alice</h2>
    <p>Mon June 15 · 9:00 – 9:30 AM</p>
    <button type="button" @click=${() => { isOpen = false; requestUpdate() }}>Close</button>
  </bc-dialog>
`
```

## Accessibility

- Pass `aria-label` or `aria-labelledby` on `<bc-dialog>` to describe the dialog for screen readers.
- The native `<dialog>` element provides built-in focus trapping — the first focusable element inside receives focus when the dialog opens.
- Pressing **Escape** fires the native `cancel` event, which `BcDialogElement` converts to the `onBcClose` callback.

## Typical pattern: event detail modal

```ts
@customElement('my-calendar')
class MyCalendarElement extends LitElement {
  override createRenderRoot() { return this }

  private _selectedEvent = null

  private _openDetail = (event) => {
    this._selectedEvent = event
    this.requestUpdate()
  }

  private _closeDetail = () => {
    this._selectedEvent = null
    this.requestUpdate()
  }

  override render() {
    return html`
      <bc-calendar
        .localizer=${localizer}
        .events=${this._events}
        .onEventClick=${this._openDetail}
      >
        <bc-default-toolbar></bc-default-toolbar>
        <bc-month-view></bc-month-view>
        <bc-time-grid-view></bc-time-grid-view>
        <bc-agenda-view></bc-agenda-view>
      </bc-calendar>

      <bc-dialog
        .open=${!!this._selectedEvent}
        .onBcClose=${this._closeDetail}
        aria-label="Event detail"
      >
        ${this._selectedEvent
          ? html`
            <h2>${this._selectedEvent.title}</h2>
            <button type="button" @click=${this._closeDetail}>Close</button>
          `
          : ''}
      </bc-dialog>
    `
  }
}
```
