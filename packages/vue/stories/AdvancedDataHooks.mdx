# Data Composables

Two composables expose the raw view model data as render-ready arrays so you can loop over them in **fully custom headless views**. The built-in view components call these internally; they are exported so you can use them when building your own layout from scratch with `useMonthView` or `useAgendaView`.

Both composables must be called inside a component that lives inside a `CalendarProvider`. Both return `null` when a different view is currently active, so your component can return early and bow out cleanly.

```ts
import { useMonthWeeks, useAgendaRows } from '@big-calendar/vue'
```

---

## `useMonthWeeks`

Returns the month grid as a typed, render-ready structure. Use this when you want to build a custom month layout — for example, rendering the grid as a `<table>` or applying a completely different visual arrangement — without reimplementing the layout algorithm.

```vue
<script setup>
import { useMonthWeeks, EventButton } from '@big-calendar/vue'

const grid = useMonthWeeks()
</script>

<template>
  <template v-if="grid !== null">
    <div>
      <div class="weekday-row">
        <span v-for="wd in grid.weekdays" :key="wd.day">{{ wd.short }}</span>
      </div>
      <div v-for="week in grid.weeks" :key="week.key" class="week-row">
        <div v-for="day in week.days" :key="day.day" :data-today="day.isToday">
          {{ day.label }}
          <button v-if="day.extra">+{{ day.extra.count }} more</button>
        </div>
        <EventButton
          v-for="seg in week.segments"
          :key="seg.key"
          :event="seg.event"
          :title="seg.title"
          :resizeEdges="[]"
        >
          {{ seg.title }}
        </EventButton>
      </div>
    </div>
  </template>
</template>
```

## `useAgendaRows`

Returns the agenda view model as a flat list of day groups, each containing its resolved events. Use this when you want to build a fully custom agenda layout.

```vue
<script setup>
import { useAgendaRows, AgendaEventButton } from '@big-calendar/vue'

const rows = useAgendaRows()
</script>

<template>
  <template v-if="rows !== null">
    <div>
      <section v-for="row in rows" :key="row.day">
        <h2>{{ row.label }}</h2>
        <div v-for="item in row.events" :key="item.key">
          <span>{{ item.allDay ? 'All day' : item.time }}</span>
          <AgendaEventButton :event="item.event" :title="item.title" />
        </div>
      </section>
    </div>
  </template>
</template>
```
